**Валидация через Pydantic:**

```python
from pydantic import BaseModel

class TextRequest(BaseModel):
    text: str
    limit: int = 10

@app.post("/analyze")
def analyze(data: TextRequest):
    words = data.text.split()
    return {"word_count": len(words)}
```

**Возврат ошибки:**

```python
from fastapi import HTTPException

@app.get("/items/{item_id}")
def get_item(item_id: int):
    if item_id < 0:
        raise HTTPException(status_code=400, detail="ID не может быть отрицательным")
    return {"id": item_id}
```

# Список тем

Тема назначается преподавателем. У каждого студента — своя тема.

| № | Тема | Описание |
|---|---|---|
| 1 | Анализатор текста | Принимает текст, возвращает кол-во слов, символов, топ-5 слов по частоте |
| 2 | Транслитератор | Переводит текст с кириллицы на латиницу и обратно |
| 3 | Генератор Lorem Ipsum | Генерирует N абзацев или слов псевдо-текста |
| 4 | Конвертер единиц | Конвертирует длину / вес / температуру (тип через параметр) |
| 5 | Калькулятор с историей | POST для вычислений, GET для получения истории операций (in-memory) |
| 6 | Анализатор числа | Определяет: простое/составное, чётное/нечётное, возвращает список делителей |
| 7 | Счётчик дней | Считает кол-во дней между двумя датами, возраст по дате рождения |
| 8 | Конвертер часовых поясов | Принимает время и две TZ, возвращает переведённое время |
| 9 | Помощник по датам | День недели по дате, сколько дней до следующей пятницы / дня рождения |
| 10 | Base64 утилита | Кодирует и декодирует строки в Base64 |
| 11 | URL утилита | Кодирует/декодирует URL, парсит query string на составные части |
| 12 | Генератор паролей | Генерирует пароль с параметрами: длина, цифры, спецсимволы, верхний регистр |
| 13 | Валидатор данных | Проверяет email, номер телефона (RU), возвращает детали ошибки |
| 14 | UUID утилита | Генерирует UUID разных версий, валидирует переданный UUID |
| 15 | Конвертер цвета | Переводит цвет между форматами HEX, RGB, HSL |
| 16 | Markdown конвертер | Принимает Markdown-текст, возвращает HTML (через библиотеку `markdown`) |

---

> Если тема кажется слишком простой — добавь дополнительный эндпоинт или параметр. Если слишком сложной — спроси преподавателя.
